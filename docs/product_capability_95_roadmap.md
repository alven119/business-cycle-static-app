# Product Capability 95 Percent Roadmap

Phase 75 supersedes the earlier Phase 56-64 three-capability target roadmap.
The user goal is now broader: bring all eight governed product capabilities to
at least 95% of formal production use within ten phases, while preserving the
ordered-cycle product doctrine.

Percentages are governed orientation numbers, not economic validation,
portfolio advice, or a formal current phase decision.

## Scope

The Phase 75-84 roadmap advances:

1. 景氣階段判讀能力
2. 轉折風險偵測能力
3. 解釋能力
4. Portfolio policy research 能力
5. 歷史重播與回測能力
6. 安全輸出治理能力
7. 時間完整性與 abstention
8. 模型治理與前瞻驗證

It does not create a standalone current phase classifier, phase rank, phase
winner, role-count vote, current allocation recommendation, or trade action.

## Guardrails

- No standalone current phase classifier.
- No phase rank, winner, or arbitrary phase score as the product answer.
- No candidate/current phase emission before an explicit migration gate.
- No supporting proxy promoted to book-core evidence.
- No portfolio action, buy/sell signal, or investment advice.
- No historical result tuning.
- No public output unless the relevant phase explicitly opens that gate.
- No raw data, cache, book PDF, or secret commit.
