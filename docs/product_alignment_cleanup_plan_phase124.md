# Phase 124：配置研究與歷史重播互動產品面

Phase 124 將既有八個配置研究模板、五個預註冊歷史情境與 156 個月的 strict
input-readiness timeline 接到私人 NAS。這一階段完成可操作 UX，不執行模型、績效或
state transition。

## 配置研究

- 以 declared 榮景和 legal next recession 作研究脈絡。
- 顯示被動基準、股票／現金、股票／長債、榮景 70／50／30、衰退防守與復甦模板。
- 70／50／30 只標為書籍 backtest-only 參數格；不是目前配置建議或交易指令。
- 顯示 Phase 123 live transition lane 狀態，watch 不會直接觸發配置動作。

## 歷史重播

- 支援網路泡沫、金融危機、COVID、歐債疑慮與 2018 late-cycle 五個事件。
- 提供 scenario selector、PIT／revised mode switch 與月度 playhead。
- 每月顯示 strict input 完整度、應 abstain 的原因及 attribution roles。
- PIT 缺漏不得回退 revised；revised 只作 diagnostic comparison。

## 下一階段

Phase 125 會在相同產品面執行 strict PIT replay、total-return 與 cash-flow-aware
research backtest，並呈現成本、turnover、drawdown、false-signal 與 missed-recovery
cost；歷史結果不得回頭調整 evaluator 規則。
