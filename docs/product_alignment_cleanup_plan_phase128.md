# Phase 128：全循環配置研究與頁面完整度收斂

Phase 128 解決配置頁將「沒有書籍精確權重」誤呈現為「尚待 Phase 125」的問題。六個具明確數值參數的模板沿用 16 組 strict 固定參數研究結果；衰退防守與復甦再承擔風險兩個 doctrine templates 改為 evidence-context-only，避免發明書中不存在的權重。

配置頁新增四個合法轉折的研究時機矩陣：

- 榮景到衰退：榮景期已先去風險，不等待衰退確認才開始注意。
- 衰退到復甦：落底 watch 對錯失復甦成本重要，但政策寬鬆不能單獨確認。
- 復甦到成長：書籍股票比例不變，重點是確認復甦廣度能否接棒。
- 成長到榮景：一旦進入 governed boom，書籍去風險研究規則開始具有意義。

所有內容維持 research-only：watch 與 confirmation 都不自動產生配置動作，也不輸出個人化買賣指令。
